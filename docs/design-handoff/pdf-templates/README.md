# PDF-шаблоны StoryGrow — «магическое издательство»

Drop-in замены для `backend/src/pdf/page-templates/*.html`. Те же классы,
те же `{{плейсхолдеры}}`, тот же размер A5 (874×1240 px) — `pdf-render.service.ts`
менять не нужно.

## Файлы
- `cover.html` · `image-top.html` · `image-bottom.html` · `image-left.html` · `text-focus.html` · `final.html`

## Что изменилось
- Единый книжный стиль: прохладно-кремовая бумага `oklch(0.986 0.012 250)`, индиго-чернила, искры-орнаменты.
- Заголовки — **Bricolage Grotesque**, текст — **Outfit** (подключаются через `@import` в `<style>` каждого шаблона; Puppeteer ждёт `networkidle0`, шрифты успевают загрузиться).
- Обложка: полноэкранная иллюстрация, градиентный занавес снизу, ваймарк, эйрбоу «Волшебная история», искры.
- Финал: мораль как акцентная фраза + вопросы для обсуждения с нумерованными градиентными кружками.

## Превью
Открой `PDF Templates Preview.html` — все 6 шаблонов с плейсхолдерами и примером текста.

## Рекомендация (необязательно)
Для офлайн-надёжности продакшена замени `@import` Google Fonts на самостоятельный хостинг
шрифтов (`woff2`) и подключи их один раз в `buildDocument()` в `<head>`, убрав `@import` из шаблонов.
Это уберёт сетевую зависимость при рендере PDF.
